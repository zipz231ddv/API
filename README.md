# Проектування та побудова API

Цей проект реалізує CRUD операції:

GET, POST, DELETE, PATCH

### Запуск проекту

1. Клонуйте репозиторій:

    git clone your URL

2. Запустіть сервер:

    php -S localhost:8080 -t public

Сервер працює за адресою http://localhost:8080.
